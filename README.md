# SPEEDTEST

Открытый код для устройства на базе ESP32-S1.
Устройство прездназначено для измерения скорости и эффективности гребца в рамках спорта Академическая гребля.
Устройство использует модули GPS и IMU для определения своего местонахождения в пространстве.

## Architecture
OPEN SOURCE код написан в системе DDD c DI(Dependency Injection), что позволяет быстро менять модули и библиотеки,
не трогая Core код программы. Достаточно заменить **Repo** реализации под свои модули, чтобы быстро начать использовать программу.

## Modules
Базовые модули, под которые написан Main код:
- ICM20948
- NEO-M9N
- ESP32-DEVCIT-S1
- SHARP MEMORY ADAFRUIT
- PowerBank2500mwt
